Replace "banned" words with another word or symbol to keep your wordpress site clean.

== Description ==
Ensure quality content on your wordpress site with this profanity filter.

== Installation ==

1. Upload the `yd-profanity-filter` directory to the `/wp-content/plugins/` directory of your site.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure your settings from the Settings --> YourDeveloper Profanity filter options
